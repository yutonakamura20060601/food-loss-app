## 🍱 食品ロス削減メモ

ブラウザで動作する、食品の賞味期限管理とレシピ提案アプリケーション

### ✨ 主な機能

#### 📝 食品管理
- 🔤 食品名を入力して、50種類の食品データベースから提案を受け取る
- 📅 賞味期限を記録
- 📦 数量を記録
- 🏷️ カテゴリ分類（野菜、果物、肉、魚、乳製品、穀物、調味料、その他）
- 🔍 カテゴリ別にフィルタリング表示

#### 🍳 レシピ提案
- 🏠 家にある食材から作れる料理を自動提案
- 50種類以上のレシピデータベース
- ✓ 用意できる材料をハイライト表示
- ⚠️ 足りない材料も表示
- 難易度表示（簡単、中程度、難しい）

#### 📊 統計情報
- 📈 登録している食品の総数
- ⏰ 期限切れまで3日以内の食品数
- 🚨 既に期限切れの食品数

#### 💾 データ管理
- 💿 データをJSONファイルとしてエクスポート
- 📥 ローカルストレージに自動保存
- 📂 以前のバックアップデータをインポート
- 🗑️ すべてのデータをリセット

### 🚀 使い方

#### 1. ファイルから起動
1. `食品_code.html` をウェブブラウザにドラッグ＆ドロップ
2. または、ブラウザで`食品_code.html`を開く

#### 2. 食品を記録する
1. 「食品を記録する」セクションで食品名を入力（オートコンプリート対応）
2. 賞味期限を選択
3. 必要に応じて数量を入力
4. カテゴリを選択
5. 「➕ 食品を追加」ボタンをクリック

#### 3. 食品を管理する
- 登録した食品が「記録した食品」セクションに表示されます
- 期限が近い食品は黄色でハイライト
- 期限切れの食品は赤色でハイライト
- 期限順に自動ソートされます
- 「削除」ボタンで食品を削除

#### 4. レシピを提案してもらう
1. 「🍳 レシピ提案」セクションの「🔍 レシピを提案する」ボタンをクリック
2. 有効な食材から作れるレシピが提案されます
3. ✓ マーク：すぐに用意できる材料
4. マーク無し：追加で用意が必要な材料

### 📊 統計情報の見方
- **登録食品数**: 現在記録している食品の総数
- **期限切れ間近**: 3日以内に期限切れになる食品
- **期限切れ**: 既に期限が切れている食品

### 💾 データの保存と復旧
- **自動保存**: ブラウザのローカルストレージに自動保存
- **エクスポート**: 「💾 データをエクスポート」で外付けHDDなどにバックアップ
- **インポート**: 「📥 データをインポート」で以前のバックアップを復旧

### ⌨️ キーボードショートカット
- `Ctrl/Cmd + N`: 食品入力欄にフォーカス
- `Ctrl/Cmd + S`: データをエクスポート
- `Enter`: 食品を追加

### 💡 使用例

**シナリオ1: 毎日の食品管理**
1. 朝ごはん後、新しく買った食品を登録
2. 夜にレシピ提案を見て、夕食の献立を決定
3. 食事が終わったら、使った食品を削除

**シナリオ2: 週末の食事計画**
1. 冷蔵庫の中にある食品をすべて登録
2. レシピ提案を見て、今週の献立を計画
3. 必要に応じて食品を追加購入

**シナリオ3: 節約と環境配慮**
1. 期限切れ間近の食品を優先的に使用
2. 食品ロスを最小化
3. 環境への貢献をしながら家計も節約

### 📚 含まれるデータベース

#### 食品データベース (50種類)
- **野菜**: トマト、キュウリ、レタス、ニンジン、ジャガイモ、タマネギ、ブロッコリー、キャベツ、ほうれん草、ナス、ピーマン、モヤシ、キノコ、トウモロコシ、ニンニク
- **果物**: バナナ、リンゴ、オレンジ、ブドウ、イチゴ、スイカ、メロン、レモン、キウイ、マンゴー
- **肉**: 鶏肉、牛肉、豚肉、ハム、ベーコン、ソーセージ、鶏卵、挽き肉
- **魚**: サーモン、マグロ、エビ、イカ、アジ
- **乳製品**: 牛乳、ヨーグルト、チーズ、バター、クリーム
- **穀物**: パン、ご飯、麺、シリアル、小麦粉、オートミール、パスタ
- **調味料**: しょう油、マヨネーズ、ケチャップ、塩、砂糖

#### レシピデータベース (50種類以上)
- **野菜系**: サラダ、スープ、カレー、シチュー、野菜炒め、和風パスタ、ラタトゥイユなど
- **肉系**: ハンバーグ、ステーキ、から揚げ、照り焼きチキン、ミートボールなど
- **魚系**: サーモングリル、エビ炒め、アジのフライ、パエリアなど
- **その他**: ピザ、オムライス、チャーハン、ラーメン、うどんなど
